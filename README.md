# 🎮 Smart Guesser Game (Java)

## 📌 Overview
Smart Guesser Game is a console-based multiplayer number guessing game developed using Core Java.  
The system generates a random number, and players try to guess it within limited attempts.

---

## 🚀 Features
- 🎯 Random number generation (1–100)
- 👥 Multiple players support
- 🔁 Limited attempts (5 per player)
- 💡 Hints system (Too High / Too Low)
- 🧮 Score calculation based on attempts
- 🏆 Winner declaration
- 🔄 Replay option

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Loops & Conditional Statements
- Java Collections (ArrayList)
- User Input Handling (Scanner)

---

## ▶️ How to Run

### 🔹 Step 1: Clone the repository
git clone https://github.com/rajpandey4706/smart-guesser-game-java.git

### 🔹 Step 2: Navigate to project folder
cd smart-guesser-game-java

### 🔹 Step 3: Compile the program
javac MyGuesserGame.java

### 🔹 Step 4: Run the program
java MyGuesserGame
